Comment compiler mon programme Java ?

Etape 1 : Créer un dossier pour compiler mon programme


mkdir out

Etape 2 : Compiler mon programme


javac src/co/simplon/game/puissance4/*.java -d out

Comment lancer mon programme Java ?

Etape 1 : Aller dans le dossier out


cd out

Etape 2 : Lancer le programme


java co.simplon.game.puissance4.Main
