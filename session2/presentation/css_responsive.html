Le responsive desing en image

Comment faire ?

Utiliser les media queries


            @media print {
              // Ici mes rules specifiques pour impression.
            }
            
            @media screen {
              // Ici mes rules specifiques pour affichage sur ecran
            }

En pratique


            @media screen and (max-width: 1024px) {
              // Ici mes rules specifiques pour affichage sur ecran
              // dont la taille est inferieure a 1024px.
            }
