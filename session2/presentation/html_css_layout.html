D'après le Larousse : l'étude du sens des unités linguistiques et de leurs combinaisons.

Les balises qui ont du sens

Exemples :

<p> permet de délimiter un paragraphe

<h1> permet de délimiter un "titre très important"

Les balises de mise en forme

A éviter maintenant que CSS est bien supporté

Mettre en forme avec <b>, <i>, <center>


            <!-- Bon exemple -->
            <p>La sémantique est <strong>très</strong> importante.</p>

            <!-- Mauvais exemple -->
            <p>La sémantique est <b>très</b> importante.</p>

Les balises neutres

La balise de type block <div>

La balise de type inline <span>

Depuis HTML5 : les balises structurantes

  • <header> pour l'entête
  • <main> pour le contenu
  • <section> pour délimiter une section de page
  • <article> pour du contenu autonome
  • <nav> pour faire un menu
  • <footer> pour le pied de page

Flexbox

CSS


            ul {
              display: flex;
              flex-direction: row;
              justify-content: center;
            }
            
            li {
              list-style-type: none;
              padding: 10px;
            }

CSS Grid

HTML


            <h1>Agencement de la salle</h1>

                <div class="table-one">Table 1</div>

CSS Grid

CSS #1


            .classroom {
                display: grid;
                grid-template-columns: repeat(6, 1fr);
                grid-gap: 10px;
                grid-auto-rows: 200px;

            }
            
            .classroom div {
                background-color: cadetblue;
            }
