Les conditions

L'indentation

Le mot clé if


            temperature = 10
            vetement = "T-shirt"
            
            if temperature < 15 : # début du bloc après les :
              vetement = "Pull"

Le mot clé else


            temperature = 20
            vetement = "rien"
            
            if temperature < 15 : # début du bloc après les :
              vetement = "Pull"
            
            else :
              vetement = "T-shirt"

Le mot clé elif


            age = 2

            if age < 0 :
              message = "merci d'entrer un age valide (positif)"

            # on ne rentre pas ici si on est déjà entré dans le cas d'avant
            elif age < 3 :
              message = "tarif : gratuit"
            
            elif age < 18 :
              message = "tarif : réduit"
            
            elif age > 65 :
              message = "tarif : réduit"
            
            else : # dans tous les autres cas
              message = "tarif : normal"

Plus propre en combinant les conditions


            age = 2

            if age < 0 :
              message = "merci d'entrer un age valide (positif)"

            # on ne rentre pas ici si on est déjà entré dans le cas d'avant
            elif age < 3 :
              message = "tarif : gratuit"

            elif age < 18 or age > 65 :
              message = "tarif : réduit"

            else : # dans tous les autres cas
              message = "tarif : normal"

Les fonctions

Mais pourquoi donc ?

Une fonction est un bout de code qui a pour but d'être réutilisé.

Elle permettra de rendre le code modulaire

Rappel sur l'exemple de la recette


            debut
              remplir_casserole(eau)
              tant_que (eau_non_bouillante)
                chauffer_casserole(feu_max)
              fin_tant_que
            
              remplir_casserole(pates)
              tant_que (pates_non_cuites)
                chauffer_casserole(feu_vif)
              fin_tant_que
            
              egouter_pates
              servir_pates
            fin

Chaque ligne est une action a redécouper en actions plus précises.

Chaque fonction peut être réutilisée avec des arguments différents.

Comment faire en Python ?


            # Définition de la fonction
            def definir_prenom():
              prenom = "Jules"
            
            # Appel de la fonction
            definir_prenom()

Les fonctions deviennent intéressantes lorsque :

  • Elle définit des paramètres
  • Elle renvoit une valeur

            def dire_bonjour(prenom):
              return "Bonjour " + prenom
            
            # Ici on range le retour de la fonction
            # "Bonjour Jules" dans la variable message_bienvenue
